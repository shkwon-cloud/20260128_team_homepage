# Project Issues

## [ISSUE-001] "프로젝트 문의" 버튼 동작 불명확

### 현상
- 메인 화면(Hero Section)에 있는 "프로젝트 문의" 버튼을 클릭해도 아무런 반응이 없음.

### 기대 결과
- 버튼을 클릭하면 "전화: 123-5678"이라는 정보가 화면에 나타나야 함.

### 분석
- `index.html`의 버튼 요소에 `id`나 `onclick` 속성이 정의되어 있지 않음.
- `index.js`에서도 해당 버튼에 대한 이벤트 리스너가 등록되어 있지 않음.

### 해결 방안 (제안)
1. 버튼에 `id="contact-btn"` 속성을 추가함.
2. `index.js`에서 해당 `id`를 참조하여 클릭 이벤트를 추가하고, 정보를 표시하는 로직(예: alert 또는 모달)을 구현함.
